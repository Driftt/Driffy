# Driffy
Giphy for Drift

Requires a Giphy API key that can be created at: https://developers.giphy.com/

## Deploying to Heroku

### CLI install 

1. `heroku create`

2. `git push heroku master`

3. `heroku config:set BOT_API_TOKEN={OAuth Access token}`

4. `heroku config:set GIPHY_API_KEY={Giphy API key}`
