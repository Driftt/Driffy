# Driffy
Giphy for Drift

## Deploying to Heroku

### CLI install 

1. `heroku create`

2. `git push heroku master`

3. `heroku config:set BOT_API_TOKEN={BOT_API_TOKEN}`

4. `heroku config:set GIPHY_API_KEY={GIPHY_API_KEY}`
